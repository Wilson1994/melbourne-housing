# melbourne-housing

Опис проєкту
Цей проєкт присвячений прогнозуванню цін на житло в Мельбурні на основі реальних даних.  
Мета побудувати модель машинного навчання, яка передбачає ціну будинку за його характеристиками.

Етапи роботи

1. Очищення даних
- Видалено зайві або технічні стовпці
- Оброблено пропуски (заповнення або заміна на NaN)
- Перетворено дату продажу у формат datetime
- Виявлено та виправлено некоректні значення (наприклад, рік побудови)
- Видалено дублікати

2. Аналіз даних (EDA)
- Розподіл цін виявився сильно скошеним вправо
- Найдорожчі будинки мають велику площу та меншу відстань до центру
- Виявлено сильну кореляцію між ціною та:
  - кількістю кімнат
  - площею житла
  - відстанню до центру
- Ознаки Rooms і Bedroom2 є майже однаковими

3. Feature Engineering
- Додано вік будинку
- Виділено рік і місяць продажу
- Створено індикатори наявності даних
- Виконано частотне кодування району (Suburb)
- Видалено зайві текстові стовпці

4. Моделювання
Було протестовано 3 моделі:
- Baseline (середнє значення)
- Linear Regression
- Random Forest Regressor


Результати

Найкращою моделлю виявився Random Forest Regressor.

Висновки
Модель здатна відносно точно прогнозувати ціни на житло, однак має труднощі з дуже дорогими об’єктами. Найважливішими факторами впливу є площа житла, кількість кімнат, район і відстань до центру.


Можливі покращення
- Використати Gradient Boosting (XGBoost / LightGBM)
- Застосувати логарифмування ціни
- Більш глибоке кодування категоріальних змінних
- Видалення або об’єднання слабких ознак Melbourne Housing Price Prediction
